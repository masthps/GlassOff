# GlassOff

Projeto final acadêmico desenvolvido para a disciplina de MWEB.

## O que é o GlassOff?

GlassOff é um e-commerce fictício de streetwear, criado com o objetivo de simular uma loja virtual moderna, responsiva e com identidade visual própria.

A proposta da marca é desenvolver peças autênticas, com foco em conforto, durabilidade e expressão individual.

## Sobre o projeto

O projeto foi desenvolvido em grupo como parte da disciplina de MWEB, com foco na criação de uma interface web completa utilizando tecnologias front-end.

O site conta com páginas institucionais e comerciais, simulando a experiência de navegação de uma loja online, desde a visualização dos produtos até a página de detalhes e contato com a marca.

## Páginas desenvolvidas

- Home Page
- Página de listagem de produtos
- Página genérica de produto
- Página About Us
- Página Contact Us
- Carrinho de compras

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- jQuery
- Font Awesome
- Design responsivo

## Funcionalidades

- Layout responsivo para desktop, tablet e celular
- Carrossel na página inicial
- Listagem de produtos
- Página de detalhes do produto
- Carrinho de compras
- Menu de navegação
- Formulário de contato
- Organização visual baseada em identidade streetwear
- Uso de variáveis CSS para facilitar manutenção das cores do projeto

## Objetivos de aprendizado

Durante o desenvolvimento do GlassOff, foram trabalhados conceitos como:

- Estruturação de páginas HTML
- Organização de estilos com CSS
- Responsividade
- Manipulação de elementos com JavaScript e jQuery
- Criação de identidade visual para uma marca fictícia
- Desenvolvimento de interface para e-commerce
- Trabalho em equipe e divisão de tarefas
